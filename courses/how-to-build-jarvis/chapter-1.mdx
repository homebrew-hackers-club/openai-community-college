## Write Your First Script That Summons AI ‚ú®

Time for the moment of truth! Let's write your first script that talks to AI.

Create a file called `first_contact.py`:

```python
import os
from openai import OpenAI
from dotenv import load_dotenv

# Load your secret credentials
load_dotenv()

# Initialize your AI connection
client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))

def summon_ai(message):
    """
    Summon AI to respond to your message
    This is where the magic happens! ü™Ñ
    """
    try:
        response = client.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=[
                {"role": "system", "content": "You are Jarvis, Tony Stark's AI assistant. Be helpful but with a bit of wit."},
                {"role": "user", "content": message}
            ],
            max_tokens=150
        )
        
        return response.choices[0].message.content
        
    except Exception as e:
        return f"Error in the matrix: {e}"

# Test your AI connection
if __name__ == "__main__":
    print("ü§ñ Initializing Jarvis...")
    print("=" * 50)
    
    # Your first message to AI
    user_message = "Hello Jarvis, introduce yourself!"
    print(f"You: {user_message}")
    
    # Get AI response
    ai_response = summon_ai(user_message)
    print(f"Jarvis: {ai_response}")
    
    print("\nüéâ First contact established! You're officially a digital wizard!")
```

## Troubleshooting Common Issues üîß

**"ModuleNotFoundError: No module named 'openai'"**
- Solution: Run `pip install openai python-dotenv pyttsx3` again

**"AuthenticationError: Incorrect API key"**
- Solution: Double-check your API key in the `.env` file

**"RateLimitError: You exceeded your current quota"**
- Solution: You might need to add billing info to your OpenAI account

**Script runs but no output:**
- Solution: Make sure your `.env` file is in the same folder as your Python script
